# Paper-improvement-Skill

A reusable ChatGPT Skill for improving academic papers end to end: critique, revision planning, section rewrites, reviewer response support, empirical integrity checks, and venue retargeting.

OpenAI describes Skills as reusable workflow bundles that can be installed from the Skills page and then used automatically when relevant. Skills are currently available in beta for Business, Enterprise, Edu, Teachers, and Healthcare workspaces. (help.openai.com)


---

Install

1. Download skill.zip.


2. Open ChatGPT.


3. Click your profile icon.


4. Open Skills.


5. Click New skill.


6. Choose Upload from your computer.


7. Upload skill.zip.


8. Install it if prompted.



OpenAI’s current help documentation says uploaded skills are managed from the Skills page and can be installed for use in ChatGPT. (help.openai.com)


---

Use

After installation, start a normal chat and ask ChatGPT to improve a paper-related artifact.

ChatGPT may use relevant skills automatically, and in supported product experiences a skill may also be explicitly selected or invoked. (help.openai.com)

Example prompts

Full critique + revision plan

> Critique this manuscript for submission to NeurIPS. Give me:

1. the 10 highest-priority weaknesses,


2. a ranked revision plan,


3. an improved abstract,


4. a claim-evidence table,


5. a submission-readiness verdict.





Rewrite core sections

> Use my paper-improvement skill on this draft and rewrite the abstract and introduction for:

sharper novelty,

clearer contributions,

tighter motivation,

better related-work positioning.




Reviewer response support

> Use my paper-improvement skill to turn these reviewer comments into a reviewer-response matrix and draft point-by-point rebuttal text.



Venue retargeting

> Retarget this paper from a machine learning conference to a clinical journal. Tell me what must change in framing, reporting, limitations, and abstract style.



Empirical integrity check

> Check whether all empirical claims in this paper are supported by the reported evidence. Flag unsupported claims, missing baselines, weak ablations, and reproducibility gaps.

Troubleshooting

I do not see Skills

Your workspace may not have Skills enabled. OpenAI says Skills availability and publishing/install controls may depend on workspace plan and admin settings. (help.openai.com)

The skill does not trigger automatically

Try naming the task explicitly in your prompt, for example:

> Use my paper-improvement skill to critique this paper and produce a revision plan.



The output is too general

Attach more context:

target venue

reviewer comments

checklist

paper PDF instead of pasted excerpts

desired output format
